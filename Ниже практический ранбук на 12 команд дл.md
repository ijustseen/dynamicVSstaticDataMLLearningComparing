Ниже практический ранбук на 12 команд для первого полного прогона.

Важно перед командой 6

- После команды 5 вручную скопируйте CK+ в experiment/data/CK+ так, чтобы были папки cohn-kanade-images и Emotion (как в [experiment/README.md](experiment/README.md#L12)).

Команды

1. cd /Users/andrew/www/maturski/experiment
2. python3 -m venv .venv
3. source .venv/bin/activate
4. python -m pip install --upgrade pip
5. pip install -r requirements.txt
6. find data/CK+ -maxdepth 2 -type d | head -n 20
7. python prepare_data.py
8. ls -la data/processed && wc -l data/processed/splits.txt
9. python train.py --model static --epochs 15
10. python train.py --model dynamic --epochs 15
11. python evaluate.py
12. ls -la checkpoints results && python -m json.tool results/evaluation_results.json | head -n 80

Что должно получиться к концу

- Есть чекпоинты: best_static.pth и best_dynamic.pth в experiment/checkpoints.
- Есть артефакты оценки в experiment/results, включая evaluation_results.json и confusion матрицы.
- Есть численное сравнение static vs dynamic по accuracy, precision, recall, f1 и fps из [experiment/evaluate.py](experiment/evaluate.py#L1).

Если останется время

- Запуск демо на камере:
  python realtime.py --model static
  python realtime.py --model dynamic
  Скрипт: [experiment/realtime.py](experiment/realtime.py#L1).

Частые сбои и быстрые фиксы

- Ошибка про splits.txt: значит не прошел prepare_data или CK+ лежит не там. Смотрите проверку в [experiment/train.py](experiment/train.py#L142).
- Очень медленно на Mac: сейчас выбор устройства только CUDA/CPU в [experiment/config.py](experiment/config.py#L65), поэтому обучение может идти на CPU.
